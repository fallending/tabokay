# Tabokay Pages

GitHub Pages site for the [Tabokay](https://github.com/fallending/tabokay) Chrome extension — privacy policy and support links.

**Live site:** https://fallending.github.io/tabokay/

## Files

| File | Description |
|------|-------------|
| [`index.md`](./index.md) | Combined privacy policy (GitHub Pages home) |
| [`PRIVACY.md`](./PRIVACY.md) | English privacy policy |
| [`PRIVACY_cn.md`](./PRIVACY_cn.md) | Simplified Chinese privacy policy |

## GitHub Pages setup

1. Repository **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` · Folder: **/ (root)**
4. Save — site publishes at `https://fallending.github.io/tabokay/`

Use that URL as the Chrome Web Store **Privacy policy URL**.
